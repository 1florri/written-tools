# 桌面草稿小组件（WRITE_TOOL）

## 功能

- 固定尺寸 `400 x 350`
- 顶部控制栏支持窗口拖动
- 置顶开关（`📌 置顶 / 📌 已置顶`）
- 四色笔刷（黑/红/蓝/绿），点击进入草稿状态，再点同色退出
- 笔刷粗细切换（`1px / 2px / 3px`）
- 橡皮擦（局部擦除，可与粗细、撤销/重做配合）
- 撤销/重做（支持撤销或重做绘制与清空操作）
- 快捷键：`Ctrl + Z` 撤销，`Ctrl + Y` / `Ctrl + Shift + Z` 重做
- 支持窗口边缘缩放（最小 `340 x 280`），画布随窗口自适应
- 草稿状态下防穿透书写（仅画布区域拦截鼠标事件）
- `ESC`：退出草稿状态并自动清屏
- `Ctrl + S`：保存当前草稿为 PNG（强制白底，默认文件名含时间戳）
- 清除全部按钮：随时清空画布
- 保存成功提示（右上角 3 秒）

## 运行

1. 安装 Node.js（建议 18+）
2. 在本目录执行：

```bash
npm install
npm run start
```

## 文件结构

- `main.js`：Electron 主进程（窗口创建、置顶切换、系统保存对话框）
- `preload.js`：安全桥接 API
- `index.html`：界面结构
- `styles.css`：视觉样式与交互状态
- `renderer.js`：画布书写、快捷键、状态管理逻辑
